sitio web para las Etiquetas Básicas de HTML,atributos y valores

HTML es un lenguaje de marcas compuesto por un conjunto de elementos que son la base de su estructura. < br > Estos elementos conforman una caja de herramientas que los autores pueden utilizar para dar forma a sus documentos. < br > Por ejemplo, puedes encontrar elementos que insertan párrafos, videos u otros documentos, o elementos que marcan citas, < br > texto importante o los títulos de libros, películas, trabajos y otras obras.
HTML define un total de 91 etiquetas, de las cuales 10 se consideran obsoletas. < br > Sin embargo, una etiqueta por sí sola a veces no contiene la suficiente información para estar completamente definida.< br > Para ello contamos con los atributos: pares nombre-valor separados por "=" y escritos en la etiqueta inicial < br > de un elemento después del nombre del elemento. El valor puede estar encerrado entre "comillas dobles" o 'simples'.< br > Existen, también, algunos atributos que afectan al elemento por su presencia en la etiqueta de inicio,< br > como puede ser el atributo ismap para el elemento< br >
